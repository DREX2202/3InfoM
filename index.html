Versionamento

O Git é um software que permite realizar versões de um projeto. O GitHub Desktop é um software que faz a ligação entre VSCode, Git e GitHub.

GitHub Desktop permite fazer a sincronia entre o repositório local e o repositório remoto

commit

É o processo de confirmação de uma versão, ou seja, a cada operação é possível confirmar a operação

Push

Este comando envia (Sincroniza) as alterações realizadas no repositório local com o repositório remoto no GitHub
